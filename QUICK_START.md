# 🚀 快速开始指南

## 5 分钟上手 DuckDuckGo + TEmail 自动验证码

### 第一步：安装扩展

1. 克隆或下载本项目
2. 打开 Chrome 浏览器，访问 `chrome://extensions/`
3. 开启右上角「开发者模式」
4. 点击「加载已解压的扩展程序」，选择项目根目录
5. **重要**：点击扩展卡片的「详细信息」，开启「在无痕模式下启用」

### 第二步：配置 DuckDuckGo（2 分钟）

1. 访问 [DuckDuckGo Email Protection](https://duckduckgo.com/email/)
2. 登录你的账号
3. 进入设置，生成 API Token
4. 复制 Token

### 第三步：配置 TEmail（1 分钟）

**使用默认配置（推荐）**：

默认配置已经可用，无需额外设置：
- 服务器：`https://email.chaijz.top`
- 邮箱：`dkdkgo@chaijz.top`

你只需要获取认证信息（联系管理员获取 JWT Token 或 Admin 密码）。

### 第四步：配置插件（1 分钟）

1. 点击浏览器工具栏的扩展图标
2. 选择「DuckDuckGo」模式
3. 粘贴 DuckDuckGo Token，点击「保存」
4. 展开「🚀 TEmail 自动验证码配置」
5. 输入 JWT Token 或 Admin 密码，点击「保存配置」

### 第五步：开始注册（1 分钟）

1. 设置注册数量（建议先设为 1 测试）
2. 设置并发窗口（建议 1-2）
3. 点击「开始注册」
4. 等待自动完成，无需任何手动操作！

## ✅ 验证成功

注册完成后，你会看到：

- ✓ 状态显示「注册成功」
- ✓ 历史记录中显示新账号
- ✓ 可以导出 Token（JSON/CSV）
- ✓ 可以同步到 Kiro IDE

## 🎯 进阶使用

### 批量注册

1. 设置注册数量（如 10）
2. 设置并发窗口（建议 2-3）
3. 点击「开始注册」
4. 插件会自动创建多个无痕窗口并发注册

### Token 验证

1. 注册完成后，点击「验证」按钮
2. 插件会批量验证所有 Token 状态
3. 自动标记：有效、封禁、过期、无效

### 导出账号

- **JSON 导出**：只导出有效的 Token
- **CSV 导出**：导出完整信息，包含状态

### 同步到 Kiro IDE

1. 点击历史记录中的「Kiro」按钮
2. 插件自动检测操作系统，生成对应命令
3. 在终端中粘贴执行
4. 重启 Kiro IDE

## ❓ 遇到问题？

### 提示"未配置 DuckDuckGo 或 TEmail 服务"

→ 检查是否完成了第三步和第四步的配置

### 收不到验证码

→ 检查 DuckDuckGo 是否设置了邮件转发到 TEmail

### 注册失败

→ 查看浏览器控制台日志（F12），确认错误信息
